# 布林带检测技能

## 触发条件
- 用户要求检测股票超买超卖
- 用户提到"布林带"、"Bollinger"、"异常检测"

## 工作流程

### Step 1: 确认参数
- 股票名称或代码
- 日期范围（可选，默认最近一年）

### Step 2: 调用工具
- 使用 `boll_detection` 工具
- 工具自动计算 MA20 和 ±2σ 上下轨

### Step 3: 呈现结果
- 工具返回检测摘要 + 超买/超卖详情 + 可视化图表
- 原样输出给用户

## 注意事项
- 需要至少 20 条数据才能计算布林带
- 超买：收盘价 > 上轨
- 超卖：收盘价 < 下轨
- 如果数据库中没有数据，工具会自动从 Tushare 拉取
