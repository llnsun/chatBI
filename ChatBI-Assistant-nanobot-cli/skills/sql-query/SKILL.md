# SQL 查询技能

## 触发条件
- 用户要求查询股票历史数据
- 用户要求对比多只股票
- 用户要求统计成交量、涨跌幅等

## 工作流程

### Step 1: 理解问题
- 确定需要查询的股票（名称或代码）
- 确定时间范围
- 确定需要的字段

### Step 2: 编写 SQL
- 使用 `exc_sql` 工具执行查询
- 表名：`stock_history`
- 中文字段名：交易日期、股票名称、股票代码、开盘价、最高价、最低价、收盘价、昨收价、涨跌额、涨跌幅、成交量、成交额

### Step 3: 呈现结果
- 工具会自动返回表格和图表
- 原样输出给用户，不要省略

## 注意事项
- 交易日期格式：YYYYMMDD
- 如果数据库中没有数据，工具会自动从 Tushare 拉取
- 多股票对比时，工具会自动分组显示
